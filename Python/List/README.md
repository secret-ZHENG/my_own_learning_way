# Singly-linked List
1. List API
   PushFront(Key) : add to front : O(1)        
   Key TopFront() : return front item   
   PopFront() : remove front item  : O(1)   
   PushBack(Key)/Append(Key) : add to back  O(n)   
   Key TopBack() : return back item   
   PopBack() : remove back item : O(n)      
   Boolean Find(Key)   : is key in list?   
   Rease(Key)  : remove key from list   
   Boolean Empty()  :  empty list?   
   AddBefore/AddAfter (Node, Key) : adds key before/after node
   Find(Key) : O(n)
